    Implicits for turning x.percent (where x is an Int or Double) into a Double scaled to where 1.0 is 100 percent.

    Example: 

    1. 1.percent == 0.01
100.percent == 1.0
99.9.percent == 0.999
500.percent == 5.0
-10.percent == -0.1
    Note

    Negative values, fractional values, and values greater than 100 are permitted.
